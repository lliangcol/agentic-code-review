# Support

## Getting Help

Use GitHub Issues for bugs, feature requests, unclear documentation, and adoption questions.

Include your runtime or surface, operating system, installation path, and the exact prompt that triggered the problem. For runtime-specific issues, state whether you used Codex, Claude Code, or both.

## Not Supported

This repository does not provide hosted review bots, model services, or project-specific private policy consulting.

Optional `review-fix-loop` issues should be reproduced against that project when they are caused by its CLI or run record behavior.
