# 来源刷新清单

刷新本 Skill 的公开依据时使用本清单。

## 范围

- 确认来源文章仍可通过已记录 URL 访问。
- 保持转述方式。不要复制或再分发文章正文。
- 更新统计或论点时，保留 vendor-research caveats。
- 同步更新英文和简体中文 source notes。
- 当刷新后的 source notes 暗示新的 Skill 行为时，同时更新英文和简体中文 implementation-analysis 文件。

## 需要复核的证据

- Review-capacity 和 churn 指标。
- Agent-authored PR rejection 或 abandonment 证据。
- AI reviewer benchmark 或 field-study claims。
- Human-on-the-loop audit guidance。
- CI、gate、prompt-injection 和 tool-action risk guidance。
- Claude Code skill discovery、`SKILL.md`、`.claude/skills` 和 `CLAUDE.md` memory guidance。

## 输出

记录：

- 刷新日期。
- 已检查的来源 URL。
- 已做变更。
- 有意保持不变的 claims。
- 后续 implementation work，如有。
