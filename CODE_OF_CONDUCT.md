# Code of Conduct

## Expected Behavior

Be respectful, concrete, and evidence-oriented when discussing review workflows, risks, and defects.

Critique code, documentation, and process decisions without personal attacks.

## Unacceptable Behavior

Harassment, threats, discriminatory language, public exposure of private information, and bad-faith disruption are not acceptable.

## Enforcement

Maintainers may edit, hide, lock, or remove contributions that violate this code of conduct.
